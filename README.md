Smart Study Management System

A Python Tkinter-based study management application designed to help students organize tasks, manage notes, and track quiz performance through an interactive graphical user interface.

Features

- User Login System
- Task Management with Live Counter
- Notes Section with Save Feature
- Quiz System with Score Tracking
- Interactive Dashboard

Technologies Used

- Python
- Tkinter
